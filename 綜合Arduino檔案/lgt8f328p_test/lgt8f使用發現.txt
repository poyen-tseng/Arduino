名稱重疊的腳位----要同時給high才會都high----
又或者說，不應該同時設定說要"使用該腳位"
不然一高一低會變0.75V  那是直接短路該腳位!!!(短路電流70mA呵呵)
因為有定義output但未定義狀態的腳位，會莫認為LOW並且具有接地的作用

可以透過
  CLKPR=0x80;
  CLKPR=0x06;//32分之一
來降頻!
而且滿準的，並且不受VIN影響

使用
#include <avr/sleep.h>

void setup() {
  CLKPR = 0x80;
  CLKPR = 0x01;
}

void loop() {                
  set_sleep_mode(SLEEP_MODE_PWR_DOWN);
  sleep_enable();
  sleep_cpu();
}
進入的休眠模式仍有1mA的待機電流，移除掉ams1117以後，待機電流剩下0.2mA(2.4V)、0.78mA(3.4V)
如果關掉ADC和BOD，待機電流剩下0.05ma(2.4V)、0.06ma(3.3V)
